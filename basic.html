BASIC

Basic Syntax

  • A PHP script can be placed anywhere in the document.

  • A PHP script starts with <?php and ends with ?>

  • The default file extension for PHP files is ".php".

        
SYNTAX:

        <?php
        // PHP code goes here
        ?>

Example:

        <?php
        echo "Hello World!";
        ?>

EMBEDDING PHP CODE IN WEB PAGE

  • To embed PHP code in an HTML page, simply include the PHP tags within the HTML code.

  • PHP interpreter processes the PHP code on the server, and the output is sent to the client as HTML:

  • A PHP script can be placed anywhere in the document.

EXAMPLE:

            <html>
                <body>
                
                <h1>My first PHP page
                
                <?php
                echo "Hello World!";
                ?>
                
                </body>
                </html>
