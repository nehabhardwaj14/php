PHP FORMS

  • HTML forms are used to collect user input, which can then be processed using PHP. This section covers the essential aspects of handling forms.

FORM CONTROLS

Common form controls include:

  • <input>: For text, password, checkbox, radio buttons, etc.

  • <select>: Dropdown lists.

  • <textarea>: Multi-line text input.

EXAMPLE:

        <form action="process.php" method="post">
            Name: <input type="text" name="name"><br>
            Age: <input type="number" name="age"><br>
            Favorite Fruit: 
            <select name="fruit">
                <option value="apple">Apple</option>
                <option value="banana">Banana</option>
                <option value="cherry">Cherry</option>
            </select><br>
            <input type="submit" value="Submit">
        </form>

SUPERGLOBAL VARIABLES

  • Superglobal variables are built-in global arrays in PHP that provide access to various data:

  • $_GET: For retrieving data sent via URL parameters.

  • $_POST: For retrieving data sent via form submissions.

  • $_REQUEST: Combines data from both $_GET and $_POST.

  • $_SESSION: For session variables.

  • $_COOKIE: For cookie variables.

SUERGLOBAL ARRAYS

  • Superglobal arrays allow easy access to user input and server information:

        <?php
        // Assuming the form is submitted
        if ($_SERVER["REQUEST_METHOD"] == "POST") {
            $name = $_POST["name"];
            $age = $_POST["age"];
            $fruit = $_POST["fruit"];
            echo "Name: $name, Age: $age, Favorite Fruit: $fruit";
        }
        ?>

IMPORTING USER INPUT

  • User input can be imported using the $_POST or $_GET superglobal arrays. For instance:

            <form action="" method="post">
                Name: <input type="text" name="name"><br>
                <input type="submit" value="Submit">
            </form>
    
            <?php
            if ($_SERVER["REQUEST_METHOD"] == "POST") {
                $name = htmlspecialchars($_POST["name"]); // Sanitizing input
                echo "Hello, $name!";
            }
            ?>

ACCESSING USER INPUT

  • Access user input using the superglobal arrays. Always sanitize user input to prevent security vulnerabilities:

        <?php
        if ($_SERVER["REQUEST_METHOD"] == "POST") {
            $age = intval($_POST["age"]); // Convert to integer
            if ($age < 18) {
                echo "You are a minor.";
            } else {
                echo "You are an adult.";
            }
        }
        ?>
