ARRAYS

An array stores multiple values in one single variable:

An array is a special variable that can hold many values under a single name, and you can access the values by referring to an index number or name.

ARRAY TYPES

  • Indexed arrays:Arrays with a numeric index

  • Associative arrays: Arrays with named keys

  • Multidimensional arrays: Arrays containing one or more arrays

INDEXED ARRAY

  • In PHP, the array elements may be a collection of key-value pairs or it may contain values only.

  • If the array consists of values only, it is said to be an indexed array, as each element is identified by an incrementing index, starting with "0".

  • An indexed array in PHP may be created either by using the array() function or with the square bracket syntax.

  • Each element in the array has a positional index, the first element being at index "0"

CREATING:

            $arr1 = array("a", 10, 9.99, true);
            $arr2 = ["a", 10, 9.99, true];

ACCESSING:

                $cars = array("Volvo", "BMW", "Toyota");
                echo $cars[0];

CHANGE VALUE:

                $cars = array("Volvo", "BMW", "Toyota");
                $cars[1] = "Ford";
                var_dump($cars);

ITERATING:

                $cars = array("Volvo", "BMW", "Toyota");
                foreach ($cars as $x) {
                  echo "$x <br>";
                }

ASSOCIATIVE ARRAYS

  • Associative arrays are arrays that use named keys that you assign to them.

  • Associative arrays are used to implement data structures such as dictionary, maps, trees, etc.

  • In PHP, the "=>" symbol is used to establish association between a key and its value.

CREATING:

                $car = array("brand"=>"Ford", "model"=>"Mustang", "year"=>1964);

ACCESSING:

                $car = array("brand"=>"Ford", "model"=>"Mustang", "year"=>1964);
                echo $car["model"];

CHANGE VALUE:

                $car = array("brand"=>"Ford", "model"=>"Mustang", "year"=>1964);
                $car["year"] = 2024;
                var_dump($car);

ITERATING:

                $car = array("brand"=>"Ford", "model"=>"Mustang", "year"=>1964)
                foreach ($car as $x => $y) {
                  echo "$x: $y <br>";
                }

MULTIDIMENSIONAL ARRAY

  • A multidimensional array is an array of arrays.

  • A PHP array may be a two-dimensional associative array also, where each element of the outer array is key-value pair, the value being another associative array.

CREATING:

            $arr = [
            [1,2,3,4],
            [10, 20, 30, 40],
            [100, 200, 300, 400]
         ];

ACCESSING:

                <?php
                $tbl = [[1,2,3,4], [10, 20, 30, 40], [100, 200, 300, 400]];    
             
                # prints number in index 2 of the row 2
                print ("Value at [2], [2] :" . $tbl[2][2]);
             ?>

ITERATING:

                <?php
                $tbl = [
                   [1,2,3,4],
                   [10, 20, 30, 40],
                   [100, 200, 300, 400]
                ];    
                echo ("\n");
                foreach ($tbl as $row){
                   foreach ($row as $elem){
                      $val = sprintf("%5d", $elem);
                      echo $val;
                   }
                   echo "\n";
                }
             ?>

DELETING ARRAY

  • To remove an existing item from an array, you can use the array_splice() function.

  • You can also use the unset() function to delete existing array items.

EXAMPLE 2: Using Splice()

            $cars = array("Volvo", "BMW", "Toyota");
            array_splice($cars, 1, 1);

EXAMPLE 2: Using unset()

            $cars = array("Volvo", "BMW", "Toyota");
            unset($cars[1]);
