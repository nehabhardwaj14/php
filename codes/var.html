VARIABLES

  • Variables are "containers" for storing information.

  • A variable can have a short name (like $x and $y) or a more descriptive name ($age, $carname, $total_volume).

RULES FOR PHP VARIABLES

  • A variable starts with the $ sign, followed by the name of the variable

  • A variable name must start with a letter or the underscore character

  • A variable name cannot start with a number

  • A variable name can only contain alpha-numeric characters and underscores (A-z, 0-9, and _ )

  • PHP variable names are case-sensitive.

SYNTAX:

        $x = "Hello, World!";

EXAMPLE:

        <?php
        $x = "Hello, World!";
        echo $x; // Outputs: Hello, World!
        ?>

SCOPE OF VARIABLES

Variable scope defines the context in which a variable is accessible.

  • The scope of a variable is the part of the script where the variable can be referenced/used.

  • PHP has four types of variable scope:

  • Local: Variables defined within a function are accessible only inside that function.

  • Global: Variables defined outside of functions can be accessed using the global keyword inside a function.

  • Static: Variables declared with the static keyword retain their value across function calls

  • Parameter: Variables passed to functions as arguments.

TYPES OF VARIABLES

PHP supports the following data types:

  • String

  • Integer

  • Float

  • Boolean

  • Array

  • Object

  • NULL

TOKENS

  • In PHP, a token is a basic unit of the language's syntax.

  • Tokens are the smallest elements in a script, representing the individual components that make up PHP code.

The types of tokens in PHP include:

  • Variables:Identified by a dollar sign ($) followed by the variable name (e.g., $variableName).

  • Operator: Symbols that represent operations (e.g., + for addition, - for subtraction).

  • Keywords:Reserved words in PHP that have special meaning (e.g., if, else, function).

  • Strings: Text data enclosed in quotes (e.g., "Hello, World!").

  • Numbers: Numeric values (e.g., 123, 3.14).

  • Symbols:Special characters that serve specific purposes (e.g., parentheses ( ), braces { }).
